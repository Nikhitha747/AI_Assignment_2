# Missionaries and Cannibals using BFS and DFS

## Problem Description
Move 3 missionaries and 3 cannibals safely across the river.

## State Representation
(M_left, C_left, Boat_position)

## Initial State
(3,3,1)

## Goal State
(0,0,0)

## Constraints
Cannibals cannot outnumber missionaries on either side.

## Algorithms Used
- Breadth First Search (BFS)
- Depth First Search (DFS)

## Data Structures
BFS → Queue
DFS → Stack

## Performance Comparison
BFS guarantees shortest path.
DFS may explore deeper paths first.
